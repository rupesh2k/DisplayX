# 🚀 Introducing DisplayX - Open Source Digital Signage

**Free, self-hosted digital signage that works on any device with a browser.**

---

## What is DisplayX?

DisplayX transforms any screen into a digital display. Perfect for retail, offices, restaurants, schools, and home use.

**Live Demo:** https://tinyurl.com/352n94m5

---

## ✨ Key Features

- **💰 Completely Free** - No subscriptions, no hidden costs
- **📴 Offline-First** - Works without internet after initial load
- **📋 Config-Driven** - Single JSON file controls everything
- **⏰ Smart Scheduling** - Time-based content playback
- **📱 Universal** - Fire TV, Android TV, Smart TVs, Raspberry Pi
- **🚫 No Backend** - Host on GitHub Pages for free

---

## 📱 Quick Start

### 1. Try the Demo
```
https://rupesh2k.github.io/DisplayX/player.html?config=https://raw.githubusercontent.com/rupesh2k/DigitalSignageContent/main/config-2026-03-27%20(1).json
```

### 2. Create Your Config
Use the visual editor:
```
https://rupesh2k.github.io/DisplayX/editor/
```

### 3. Deploy
- **Fire TV**: Open in Silk Browser
- **Raspberry Pi**: Chromium kiosk mode
- **Any device**: Just open the URL!

[Full Deployment Guide →](DEPLOYMENT-GUIDE.md)
